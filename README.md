# 🚀 TaskFlow

**TaskFlow** is a full-stack task management system with a modern Kanban-style interface designed to improve productivity and workflow organization. It provides authentication, task tracking, analytics, calendar scheduling, and export features in a smooth and responsive UI.

---

## ✨ Features

- 🔐 User Authentication (Register / Login)
- 📋 Task Management (Create, Update, Delete)
- 📊 Dashboard with Analytics & Charts
- 📅 Calendar View for Task Scheduling
- 📌 Kanban Board (Drag & Drop style workflow)
- 📈 Progress Tracking System
- 📤 Export Tasks to Excel / PDF
- ⚡ Responsive and Modern UI

---

## 🛠️ Tech Stack

**Frontend:**
- React (Vite)
- CSS3
- Axios

**Backend:**
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
