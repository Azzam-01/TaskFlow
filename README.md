# 🚀 TaskFlow

TaskFlow is a full-stack task management system with a modern Kanban-style UI designed to improve productivity and workflow organization. It provides authentication, task tracking, analytics, calendar scheduling, and export features in a smooth and responsive experience.

---

## ✨ Features

- 🔐 User Authentication (Register / Login)
- 📋 Task Management (Create, Update, Delete)
- 📌 Kanban Board Workflow
- 📊 Dashboard Analytics with Charts
- 📅 Calendar Scheduling View
- 📈 Progress Tracking System
- 📤 Export Tasks (Excel / PDF)
- ⚡ Responsive UI Design

---

## 🛠️ Tech Stack

**Frontend:**
- React (Vite)
- CSS3
- Axios

**Backend:**
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
